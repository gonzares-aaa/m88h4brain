# M88/handheld for Brain
電子辞書Brainで数字などが入力できるようにM88/handheldを改変したPC-8801エミュレータです。
SigmarionⅢ版 M88/handheld 0.06をベースにBrain用に改造したものです。

オリジナルはcisc氏作の「M88 for Win32」をTAN-Y氏がWindows CE (Sigmarion III)に移植したもので、著作権はそれぞれの作者にあります。

また、ローマ字入力機能についてはQuasi88のソースコードを一部使用しています。

オリジナルの M88/handheld に対し、以下の5つの機能を追加・修正しています。

## 追加機能
### (1)Brain用キー割り当て
キーボードは以下のようなマッピングになっています。(第3世代のキーボードを例に示します。)

### (2)サウンドミュート
外に持ち出して使う場合は、サウンドをOFFにして使いたいときが多いと思いますが、Brainでのボリューム調整は、辞書メニューから設定まで移動しなければなりません。このため、エミュレータ内に独自のミュート機能を実装しています。起動時は必ずミュートが有効な状態となります。サウンドをONにしたい場合は、起動後に、メニューから[Control]→[Mute]をタップして、ミュートを解除してください。

### (3)ローマ字入力
Quasi88のソースコードを元にローマ字入力機能を実装しています。
ローマ字はMS-IME仕様のみサポートしています。
起動時は必ずローマ字入力が有効な状態で起動します。ローマ字入力をOFFにしたい場合は、メニューから[Control]→[Romaji input]をタップするとローマ字入力が無効化されカナ入力モードになります。

### (4)どこでもセーブ
簡易的などこでもセーブ&ロード機能を実装しています。
メニューの[Control]→[Quick Save]→[Slot n] (nは1～10)を選択すると、現在の状態を`AppMain.exe`が存在するフォルダに`m88hstat.001～010`というファイル名で保存します。
なお、同名ファイルが存在した場合は、上書き確認のダイアログが表示されます。

ロードも同じように[Control]→[Quick Load]→[Slot n] (nは1～10)を選択すると`AppMain.exe`が存在するフォルダの`m88hstat.001～010`を読み込んで再開します。

### (5)その他
起動時にロケールを日本語に変更していますが、日本語フォントのパッチまでは当てていないため、日本語フォントが文字化けします。これは、ceOpenerを一度起動→終了することで回避できます。

CRTC初期化時のテキストカラーを黒から白に修正（Windows版M88 2.19aからのバックポート）しています。これにより、サザンクロスの画面が真っ黒にならずに正常に動作するようになっています。

## インストール方法
まずは、`AppMain.exe`をダウンロードし、Brainの`アプリ`フォルダ配下に適当なフォルダ名(例えば`M88hBrain`)でフォルダを作成し、そこに配置してください。
あわせて、`Index.din`, `AppMain.cfg`ファイルも配置してください。
BIOSなどのファイルは同じフォルダに配置すればOKです。

あとは、辞書メニューの追加アプリから起動できます。
このあたりの詳細は、[BrainWiki](https://brain.fandom.com/ja/wiki/%E3%82%A2%E3%83%97%E3%83%AA%E3%81%AE%E8%B5%B7%E5%8B%95%E6%96%B9%E6%B3%95)に詳しく記載されていますので、そちらをご参照ください。

なお、`gx.dll`も必要なため、[こちらの記事](https://qiita.com/nekopro2/items/ae1dd0fe8e4ee0bc23ba)などを参考に配置してください。

### ※高解像度モードで起動しない場合
0バイトの`AppMain.cfg`を置いただけでは高解像度モードで起動しない場合があります。
この場合は、PCにて`AppMain.cfg`をバイナリエディタで開き、

```
01 00 00 00 FF FF FF FF
```

と入力して保存し、このファイルに差し替えるようにしてください。
