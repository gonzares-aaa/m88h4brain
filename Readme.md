# M88/handheld for Brain
電子辞書Brainで数字などが入力できるようにM88/handheldを改変したPC-8801エミュレータです。

SigmarionⅢ版 M88/handheld 0.06をベースにBrain用に改造したものです。

オリジナルはcisc氏作の「M88 for Win32」をTAN-Y氏がWindows CE (Sigmarion III)に移植したもので、著作権はそれぞれの作者にあります。

またローマ字入力機能についてはQuasi88のソースコードを一部使用しています。

修正しているのは、以下の5つの追加機能です。

## 追加機能
### (1)Brain用キー割り当て
キーボードは以下のようなマッピングになっています。(第3世代のキーボード)

[文字切替]キー(以降は[切替]キーと表記)を押すと、一部が数字もしくは記号の配列に置き換わります。この状態で[SFT]キーを押すと、各種記号が入力できます。
なお、[記号]キーは[カナ]に, [戻る]キーは[Esc]に, [履歴/しおり]キーは[F5], [マーカーテスト]キーは[TAB], [ツール]キーは[CTRL], [HOME]キーは[CAPS Lock]にマッピングされています。

### (2)Mute機能
外に持ち出して使う場合、サウンドをOFFにして使いたいときが多いのですが、Brainの場合いちいち辞書メニューから設定まで移動して、ボリューム調整をしなければなりません。なので、ミュート機能を実装しています。
起動時は必ずミュートが有効な状態で起動します。サウンドをONにしたい場合は、起動後、メニューから「Emulate」→「Mute」をタップするとミュートが解除されます。

### (3)ローマ字入力
Quasi88のローマ字入力機能を元にローマ字入力機能を実装しています。
ローマ字はMS-IME仕様のみサポートしています。
起動時は必ずローマ字入力が有効な状態で起動します。ローマ字入力をOFFにしたい場合は、起動後メニューから[Control]→[Romaji input]をタップするとローマ字入力が無効化されカナ入力モードになります。

### (4)どこでもセーブ
簡易的などこでもセーブ&ロード機能を実装しています。適当なところでメニューの[Control]→[Quick Save]→[Slot n] (nは1～5)を選択すると、現在の状態をAppMain.exeが存在するフォルダにm88hstat.001～005というファイル名で保存されます。
なお、上書きの確認は無しに常に上書きされます。
ロードも同じように[Control]→[Quick Load]→[Slot n] (nは1～5)を選択するとAppMain.exeが存在するフォルダのm88hstat.001～005というファイルから読み込み再開されます。

### (5)その他
起動時にLocaleを日本語に変更しています。ただし、起動時に日本語フォントへパッチを当てていないので、日本語フォントが化けます。これは、ceOpenerを一度起動→終了することで回避できます。

CRTC初期化時のテキストカラーを黒から白に修正（Windows版M88 2.19aからのバックポート）しています。これにより、サザンクロスの画面が真っ黒にならずにちゃんと遊べるようになっています。

## インストール方法
まずは、AppMain.exeをダウンロードし、Brainの「アプリ」フォルダ配下に適当なフォルダ名(例えばM88Brain)でフォルダを作成し、そこに配置してください。
あわせて、Index.din, AppMain.cfgファイルも配置してください。
BIOSなどのファイルは同じフォルダに配置すればOKです。あとは、辞書メニューの追加アプリから起動できます。
このあたりのことは、[BrainWiki](https://brain.fandom.com/ja/wiki/%E3%82%A2%E3%83%97%E3%83%AA%E3%81%AE%E8%B5%B7%E5%8B%95%E6%96%B9%E6%B3%95)に詳しく記載されていますので、そちらをご参照ください。なお、gx.dllも必要なので、[このあたり](https://qiita.com/nekopro2/items/ae1dd0fe8e4ee0bc23ba)を参考に配置してください。

なお、高解像度モードで起動しない場合は、0バイトのAppMain.cfgを置いただけではダメみたいです。この場合は、PCにてAppMain.cfgをバイナリエディタで開き、

> 01 00 00 00 FF FF FF FF

と入力して保存し、このファイルに差し替えるようにしてください。
