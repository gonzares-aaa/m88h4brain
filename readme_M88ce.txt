
M88/ce (M88/pocket,M88/handheld)のソースです。
オリジナルとなるM88 for Win32 rel.2.16のソースも含まれていますので、ここに
あるソースだけでビルドが可能です。

ソースの利用、配布条件は全てM88に準拠します。
M88/ce固有部に関してもM88のものに準拠します。
M88の条項に該当する部分がM88/ce固有部にも発生する場合は、（M88をM88/ceに
読み替えて）同様の対応をお願いします。


[参考] ----------------------------------------------------------------------
【M88 のソースコードの再利用】
	・M88 のソースコードの一部，または全部を組み込んだソフトは，
	  フリーソフトとして公開することが出来ます．
	  
	・但し，src\pc88 のディレクトリの下にあるファイルを組み込む場合，
	  または商用ソフト(シェアウェア含む)へのプラグインソフトとして
	  配布する場合は，同時にそのソフトのソースコードもフリーソフトと
	  して公開ください．
	  
	・公開の際には，ドキュメント等に M88 のソースコードの一部または
	  全部を組み込んだ事と，著作権表示を明示してください．
	  また，作者への連絡を頂ければ幸いです．
	
	・M88 のソースコードを利用したソフトのソースを配布する際には，
	  M88 のソースコードのうち，そのソフトのコンパイルに必要なものに
	  限り，添付することを認めます．
	
	・商用ソフト(シェアウェア含む) に M88 のソースコードの一部，または
	  全部を組み込む際には，事前に M88 の作者の合意を得る必要があります．
	
	・M88 に改変を加えたソフトを配布する場合は，M88 の著作権表示，
	  および改変内容を明示してください．
-----------------------------------------------------------------------------

ソースコードはあるがままの形で提供します。
サポートは行えませんので、問題が発生した場合もご自分で対応をしていただくよう
お願いします。


コンパイルはembedded Visual C++ (eVC++) 3.0で行っております。
sigmarionIII用はeVC++ 4.0で行っております。
細かくは

・PocketPC用
eVC++3.0 + PocketPC SDK

・HandheldPC 2000用
eVC++3.0 + HandheldPC 2000 SDK

・sigmarionIII用
evC++4.0 + sigmarionIII SDK

です。

PocketPC用とHandheldPC用はM88P.VCWというワークスペースに両方のプロジェクトを
持っています。
sigmarionIII用はM88sig3/M88sig3.vcwです。

Visual C++ 6.0に入っているASSERT.Hをsrc/winceにコピーしてください。

基本的にオリジナルソースのクラスをWindowsCE対応をした独自クラスで継承、
実装する形で対応をしています。
オリジナルソースはなるべく手を加えないようにしていますが、どうしても
触らなければいけない部分もあります。変更内容は「変更メモ.txt」に記載しました
ので、コア部分を最新版に入れ替えることをした場合などは参考にしてください。
