【ソースコードについて】

	Visual C++ 6.0 + SP3 用でコンパイルできます.
	基本的に C++ で書かれていますが，一部ファイルではインライン
	アセンブリを使用しています．
	
	インラインアセンブリを全く使用せずにコンパイルすることも可能です．
	(Borland C++ 5.02 でコンパイルできることを確認済)
	
	M88.dsp を開けばすぐにコンパイルできるはずです。
	
	VC5 でもビルドできるかどうかはわかりません．
	(インラインアセンブリ関係なんかはダメっぽいですね)
	
	FM 音源のライト(?)バージョンを作成する場合は，
	src/devices/fmgen.h の中の FM_LIGHT を 1 にしてください．
	
	(追加)
	ビルドを行う前に，writetag.cpp を単独でコンパイルして writetag.exe を
	作成しておいてください．あるいは，『リンク後の処理』を削除してください．
